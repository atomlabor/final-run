# Final Run - Pebble Steel App

Eine spezialisierte Lauf-App für die Pebble Steel mit Ziel-Navigation und Lap-Tracking.

## Features
- **Ziel-Navigation:** Luftlinie zum Ziel (einstellbar über Clay-Settings).
- **Pace-Tracking:** Anzeige der Durchschnitts-Pace pro Kilometer.
- **Lap-History:** Speicherung der letzten Kilometer-Zeiten.
- **Auto-Pause:** Pausiert automatisch bei Stillstand.
- **Stabilisiertes GPS:** 70m Zielradius für sichere Ankunft.


Konfiguriere dein Ziel in der Pebble Handy-App (Pebble Core App) unter "Settings".
Setze dort die Längen und Breitengrade deines Ziels ein. 

Umgesetzt durch https://atomlabor.de für die Pebble Community
